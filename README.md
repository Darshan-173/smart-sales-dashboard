# 📊 Smart Dynamic Sales Dashboard

A fully dynamic Streamlit dashboard that adapts to uploaded datasets.

## Features
- Auto-detect columns (date, product, customer, quantity)
- Dynamic filters
- KPI metrics
- Top products & customers visualization
- Monthly trend analysis
- Smart insights
- Download filtered data

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
